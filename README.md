# PaisaFlow

**Track your spending. Split trips. Settle smarter.**

PaisaFlow is a modern expense tracking web app built for two real-life use cases:

1. **Daily Spend Tracking** — track personal expenses like food, travel, subscriptions, books, shopping, and bills.
2. **Trip Split Mode** — manage group trip expenses with friends and automatically calculate who owes whom.

Example:  
If Omkar pays ₹120 for soda for 3 friends, PaisaFlow calculates Omkar’s share as ₹40 and shows that the other two friends owe ₹40 each. If another friend pays for breakfast, the app adjusts the final balance automatically.

---

## ✨ Features

### Daily Spend Mode

- Add personal daily expenses
- Track amount, category, date, notes, and payment mode
- View monthly total spending
- View today’s expenses
- Category-wise expense breakdown
- Budget progress tracking
- Spending trend analytics

### Trip Split Mode

- Create trips with friends
- Add trip members
- Add group expenses
- Split expenses equally
- Track who paid and who owes
- View each member’s total paid and actual share
- See final settlement suggestions
- Mark settlements as completed

### Smart Split Calculation

PaisaFlow calculates:

- Total paid by each member
- Actual share of each member
- Net balance
- Who should pay whom
- Minimum settlement transactions

Example:

```txt
Soda Expense: ₹120
Paid by: Omkar
Members: Omkar, Friend A, Friend B

Each share: ₹40

Friend A owes Omkar ₹40
Friend B owes Omkar ₹40
Omkar's real cost: ₹40
