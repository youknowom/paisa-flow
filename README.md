## 🌟 Project Summary

**PaisaFlow** is a modern expense tracking and trip-splitting web app that helps users manage daily expenses, track monthly spending, and settle shared costs with friends.

It is built for two real-life use cases:

1. **Daily Spend Tracking** — track personal expenses like food, travel, shopping, subscriptions, rent, and bills.
2. **Trip Split Mode** — split group expenses with friends and automatically calculate who owes whom.

---

## ✨ Features

### 💸 Daily Spend Tracking

- Add and manage personal expenses
- Track amount, category, date, notes, and payment mode
- View today, weekly, and monthly spending
- Filter expenses by category and payment mode
- Track budget and spending summary

### 🧳 Trip Split Mode

- Create trips with friends
- Add members and shared expenses
- Split expenses equally
- Track who paid and who owes
- View final settlement suggestions
- Mark settlements as completed
